## Verified against the literature

**VERDICT.** Model B's electron-density elimination is **not sound**, and the method is **neither standardless nor temperature-free**. The elimination rests on two independent, fatal defects. (1) The "decoupling" of n_e invokes the First Generalized Mean Value Theorem for Integrals, which is an *existence-only* theorem (Apostol 1974, ref 21): it guarantees some coordinate ξ exists but supplies no value, so the transient function n_e(t) is merely *relabeled* as an undetermined constant n_e(ξ) — it is not removed. (2) The loop only closes by inverting the §5.1 identity that *defines* the asynchronous signal as three-body-recombination flux; that identity is itself unsupported, because Noda's asynchronous spectrum is a Hilbert-transform (dissimilarity / sequential-order) quantity, not a "Wronskian integral" or a time-derivative flux (Noda 1993, ref 20). All temperature dependence — partition functions U(T), the Saha/Boltzmann exponential exp(−E_ion/kT), the T^{3/2} factor, and the three-body scaling k_r ∝ T_e^(−9/2) — plus the absolute optical/rate calibration is then buried inside the single lumped parameter ξ, which the engine must receive **precomputed**. So the method trades an independently measurable unknown (n_e via Stark broadening) for a T- and instrument-dependent ξ that requires prior knowledge of the temperature and absolute rate constants — arguably a worse position than classical CF-LIBS. **What survives:** the zero-diagonal proof of the asynchronous matrix (correct, but see C8), the §5.3 variance≈mean-square sub-step (valid only when the gate window ≫ ionic decay time), the LTE / Saha-Eggert / Stewart–Pyatt prose framing, and the Hilbert-Noda kernel implementation (sign/orientation correct). Most references point to real papers. **What collapses:** the central kinetic bridge, the n_e elimination, the "standardless/temperature-free" claim, the "ILR" claim (the code computes plain Aitchison closure), the Cσ-graph attribution (fabricated citation), and the self-absorption argument built on the "Model C" strawman.

### FATAL

**C1 — ξ is temperature-dependent; Model B is not standardless or temperature-free.** §6 calls ξ "the Invariant Multi-Element Saha-2DCOS Parameter, grouping all remaining physical constants, atomic transitions, and partition paths," and the abstract claims Model B "analytically decouples and eliminates the n_e coordinate" for standardless quantification. Tracing the paper's own substitution chain refutes this. Saha-Eggert gives n_ion/n_atom = S(T)/n_e with S(T) = (2U_ion(T)/U_atom(T))·(2πm_e kT/h²)^{3/2}·exp(−E_ion/kT); substituting the §5.3 expression for n_e forces ξ = S(T)·√(k_r(T)·T·optical). Since k_r ∝ T_e^(−9/2) (standard three-body scaling, confirmed), ξ carries the partition functions U(T), the T^{3/2} factor, the Saha/Boltzmann exponential, the T^(−9/2) recombination scaling, *and* the absolute optical collection factor. The phrase "partition paths" already concedes U(T) lives inside ξ. A T- and instrument-dependent ξ cannot be a precomputed constant, and the engine's own symbol name `xi(T)` plus the word "Precomputed" confirm it must be supplied before the inversion — with no temperature-retrieval step anywhere in §5–§6 or the code. *Literature:* Saha ionization equation; Tognoni et al., *Spectrochim. Acta B* 65:1 (2010) [ref 5]; Hinnov & Hirschberg, Phys. Rev. 125:795 (1962) [ref 18] with k_3 ∝ T_e^(−9/2). *Correction:* drop the "temperature-free/standardless" framing; state that ξ = ξ(T, U(T), E_ion, k_r(T), optical) requires an independent temperature determination and absolute intensity/rate calibration. n_e is not eliminated but repackaged into ξ.

**C2 — the First Mean Value Theorem relabels n_e, it does not remove it.** §5.2 applies the First Generalized Mean Value Theorem for Integrals (correctly cited to Apostol) and claims it "proves the existence of an integrated mean plasma coordinate ξ that allows us to extract the terms outside the integration boundary," then §5.3 "solves explicitly for the average free electron density." The theorem is a pure *existence* statement: it guarantees ∃ξ∈(a,b) with ∫f·g = f(ξ)∫g but supplies no value for ξ. Pulling n_e² out as ⟨n_e⟩² = n_e(ξ)² therefore replaces the transient function n_e(t) with an *unknown constant* n_e(ξ) — the unknown is renamed, not removed. Worse, the subsequent "solve for ⟨n_e⟩" only closes because §5.1 *defined* the asynchronous signal ψ as optical·k_r·n_e²·∫n_ion²; algebraically inverting that to recover n_e is tautological and returns exactly the n_e that was assumed to drive ψ. No independent measurement of n_e ever enters. *Literature:* Apostol, *Mathematical Analysis* [ref 21] (existence only). *Correction:* state plainly that the MVT yields an undetermined ⟨n_e⟩ = n_e(ξ) at an unknown coordinate; any numerical value assigned to it comes from the assumed kinetic identity, not from data.

**C3 — the "ILR projection" is plain Aitchison closure.** §2.2 says concentrations are projected into the simplex "using the Isometric Log-Ratio (ILR) transformation [16]," and the engine comment repeats it. But `softmax(log x)_i = exp(log x_i)/Σ_j exp(log x_j) = x_i/Σ_j x_j`, which is exactly Aitchison's closure operator C(x) (Aitchison 1986): ordinary re-normalization onto S^D, with no dimension reduction, no log-ratio coordinates (the log and softmax's internal exp cancel identically), no orthonormal basis, and no isometry. ILR, as defined by Egozcue et al. (2003), is ilr(x) := V^T·clr(x) with V an orthonormal basis of the (D−1)-dim clr-plane; it is an isometry S^D → R^{D−1} that strictly reduces dimension. The code performs none of ILR's defining operations — `jax.nn.softmax(jnp.log(x))` is mathematically identical to `x/jnp.sum(x)`, and it was empirically confirmed to sum to exactly 1.0. This is decidable by algebra alone; no missing equation is involved. *Literature:* Egozcue, Pawlowsky-Glahn, Mateu-Figueras & Barceló-Vidal, *Math. Geol.* 35(3):279–300 (2003) [ref 16]; Aitchison, *The Statistical Analysis of Compositional Data* (1986) [ref 15]. *Correction:* relabel the operation as compositional closure C(x) in both the code comment and §2.2; the geometric benefits ILR would confer (Aitchison-metric-respecting gradients, subcompositional coherence) are *not* obtained. A genuine ILR needs an unconstrained z∈R^{D−1}, a fixed orthonormal basis V, and ilr⁻¹(z)=C(exp(Vz)).

**C4 — the Cσ-graph citation is fabricated and misattributed.** §1.2 credits the "C-Sigma Graph method" to "Corsi et al. (2000)" (ref 11: Appl. Spectrosc. 54(4) 623–633, DOI 10.1366/0003702001949717). No such paper exists. The exact title "CSigma graphs: A new approach for plasma characterization in laser-induced breakdown spectroscopy" belongs to C. Aragón & J. A. Aguilera, *JQSRT* 149 (2014) 90–102 (corrigendum JQSRT 155, 2015); the conceptual origin is Aguilera & Aragón (2007). The cited page range does not exist (vol 54 pp. 624–629 is Mullins et al.), and the supplied DOI resolves to Panne et al., Appl. Spectrosc. 54(4) 536–547. The monograph's claim that Corsi et al. "elegantly model self-absorption via the Curve of Growth" via Cσ graphs is therefore unsupported by any real source. *Literature:* Aragón & Aguilera, *JQSRT* 149:90–102 (2014); Aguilera & Aragón, *Spectrochim. Acta B* (2007). *Correction:* delete the fabricated Corsi et al. (2000) entry and attribute the method to Aragón & Aguilera (2014).

### MAJOR

**C5 — "Noda showed async = a Wronskian flux integral" is unsupported.** §5.1 asserts that "for monotonic, non-periodic relaxation profiles, Noda showed that the asynchronous correlation represents a temporal cross-peak Wronskian integral," then maps it to three-body-recombination flux and drops one term. Noda's asynchronous spectrum is defined via the Hilbert-Noda transform — a nonlocal Hilbert integral operator, not a derivative — and its accepted reading is dissimilarity / sequential order of intensity changes. No Noda result equates the async to a Wronskian ∫(f·g′−f′·g), and because the Hilbert transform is not a time-derivative, the async is not a flux; the async↔recombination-flux bridge is not a mathematical identity. It also silently assumes single-driver monotonic relaxation, LTE intensity∝population, unblended lines, and three-body dominance — whereas neutral birth/loss in a real expanding LPP is dominated by hydrodynamic expansion/advection plus radiative recombination. Dropping one antisymmetric term is unjustified near the ion→neutral crossover. *Literature:* Noda, *Appl. Spectrosc.* 47:1329 (1993) [ref 20]; He et al. (FFT-based 2DCOS); Nagpal et al., arXiv:2010.06017 (2020). *Correction:* remove the "Noda / Wronskian" attribution; any async ∝ flux claim needs a real derivation retaining expansion losses and competing kinetics.

**C6 — the three-body recombination prefactor is not a "universal quantum-mechanical coupling constant."** §3.1 states the recombination coefficient "is governed by the Hinnov-Hirschberg electron-temperature power-law relation," with the prefactor described as "a universal quantum-mechanical coupling constant." Three-body (e–e–ion) recombination is a *classical* result: its scaling K_3b ∝ T_e^(−9/2) (effective α ∝ n_e·T_e^(−9/2)) follows from detailed balance against Thomson electron-impact ionization plus Saha. The prefactor is not universal — it carries the ion charge-state dependence (≈Z³), a Coulomb/threshold logarithm, and reduced-mass/cross-section factors — and it is not quantum-mechanical. *Literature:* Thomson three-body theory; Stevefelt, Boulmer & Delpech, Phys. Rev. A 12:1246 (1975) (collisional-radiative recombination); Mansbach & Keck, Phys. Rev. 181:275 (1969); T_e^(−9/2) confirmed in Fletcher, Zhang & Rolston, PRL 99:145001 (2007). *(These two source attributions were themselves slightly off in the first-pass audit and were corrected during the rebuild's F5 citation verification.)* *Correction:* describe the prefactor as a charge-state- and Coulomb-log-dependent classical quantity, and state the K_3b ∝ T_e^(−9/2) scaling explicitly, attributing it to classical detailed-balance/Thomson kinetics.

**C7 — Colgan et al. (2014) does not support the geological/high-Z composition-error claim.** §1.2 attributes "systematic temperature overestimations and composition errors up to ⟦?⟧% in complex geological and high-Z matrices" to ref 9 (Colgan et al. 2014). That paper is an ab-initio (Hartree–Fock / collisional-radiative) calculation of the emission spectrum of a *pure* iron-oxide (Fe₂O₃) LIBS plasma, benchmarking Fe I/Fe II line intensities. Fe (Z=26) is a single-element, moderate-Z sample; the paper studies neither matrix effects nor inhomogeneity error, and reports no % figure of the kind claimed. *Literature:* Colgan, Judge, Kilcrease & Barefield, *Spectrochim. Acta B* 97:65–73 (2014). *Correction:* cite a source that actually quantifies isothermal-zone/inhomogeneity error (e.g., spatially resolved or inhomogeneous-plasma CF-LIBS work) and fill in a defensible number, or drop the numeric claim.

**C8 — a zero asynchronous diagonal cannot "render Model C unphysical."** §7/§4.2 prove that the async diagonal is identically zero and conclude this makes a self-absorption "Model C" unphysical. The zero diagonal is a *universal* mathematical property of Noda's antisymmetric operator, Ψ(ν₁,ν₂)=−Ψ(ν₂,ν₁) ⟹ Ψ(ν,ν)=0, true for *every* dynamic dataset regardless of physics; it would equally "falsify" every model, so it discriminates nothing. "Model C" is a strawman. *Literature:* Noda, *Appl. Spectrosc.* 47:1329 (1993) [ref 20]. *Correction:* keep the correct zero-diagonal proof but drop the inference; state that self-absorption diagnostics must therefore live in the off-diagonal cross-peaks, without claiming any competing model is disproven.

**C9 — the wing-to-center "butterfly" self-absorption signature is conjecture, not an established result.** §7.1 and the abstract present the four-quadrant "butterfly" async pattern as a known diagnostic. No time-resolved-2DCOS study of LIBS self-absorption demonstrates it; the only genuine 2DCOS-of-LIBS paper concerns SNR improvement, and time-resolved self-absorption is studied via line-shape/intensity evolution, not an asynchronous butterfly. The mechanism is physically motivatable but unvalidated, and its governing equation is a ⟦MATH⟧ gap. *Literature:* Tang et al., *Opt.
